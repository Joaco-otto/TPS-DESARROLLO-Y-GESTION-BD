Preguntas Frecuentes

1.- ¿Cuales son los pasos para registrar un dominio?
2.- ¿Debo contar con una pagina web y con el servicio de "Hosting" para registrar y delegar un dominio?
3.- ¿NIC Argentina presta servicios de Hosting, correo electronicos y otros ?

Respuestas

1- Completar el formulario de registro con los datos requeridos, incluyendo información del titular del dominio y datos de contacto.
2- Pagar la tarifa correspondiente por el registro del dominio. NIC Argentina ofrece diferentes métodos de pago.
3- Una vez procesado el pago, NIC Argentina procederá a registrar el dominio y enviará una confirmación al correo electrónico proporcionado.

No es necesario contar con una página web o servicio de hosting para registrar un dominio. El registro del dominio es independiente y puede realizarse sin tener un sitio web asociado. Sin embargo, para que el dominio funcione y sea accesible en Internet, eventualmente necesitarás un servicio de hosting donde alojar tu sitio web.

No, NIC Argentina no presta servicios de hosting, correo electrónico u otros servicios relacionados con la gestión de sitios web. Su función principal es la administración y registro de nombres de dominio bajo el código de país .ar. Para servicios adicionales como hosting o correo electrónico, deberás recurrir a proveedores especializados en esos servicios.
